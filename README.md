# 💻 Projetos de Sites

Este repositório é dedicado a hospedar diversos projetos e templates de sites em construção, variando desde páginas simples de apresentação até layouts mais complexos. 

---

## 🎯 Objetivo
O principal foco aqui é o desenvolvimento e prática contínua de Design de Interfaces (UI) e criação de sites responsivos. Cada pasta representa um projeto diferente hospedado lado a lado.

---

## 📁 Estrutura do Repositório

Aqui estão alguns dos projetos encontrados:

| Projeto/Pasta | Descrição Resumida |
|---|---|
| `Andreia/` | Desenvolvimento de template/página específica |
| `Judiciario/` | Mockup/site voltado para tema judiciário |
| `Medidor_Aura/` | Aplicação interativa focada no Medidor de Aura |
| `Silva/` | Mais um template/layout em desenvolvimento |
| `index.html` | Uma página agregadora ou hub central possivelmente para guiar até as pastas |

---

## 🚀 Como Executar
Todos os projetos são construídos utilizando tecnologias web padrão (Frontend).

1. Faça o clone do projeto:
```bash
git clone https://github.com/MaskDMoa/Sites.git
```
2. Escolha o site que deseja ver navegando pelas pastas.
3. Abra o arquivo `index.html` do respectivo site em seu navegador favorito (Google Chrome, Firefox, etc).

---
